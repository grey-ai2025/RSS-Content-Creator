# Figma Integrates OpenAI Codex — Becoming the Multi-Model Design-to-Code Hub

**Title:** Figma partners with OpenAI to bake in support for Codex

**Source URL:** https://techcrunch.com/2026/02/26/figma-partners-with-openai-to-bake-in-support-for-codex/

**Publication Date:** February 26, 2026

**Keyword Match:** AI coding tools, multi-agent systems

## Summary

Figma has partnered with OpenAI to integrate Codex, OpenAI's code-generation model, directly into the design platform — following a similar integration with Anthropic's Claude Code announced just days prior. The move positions Figma as a multi-model AI coding hub sitting at the intersection of design and engineering workflows, rather than a single-vendor AI partner. Codex integration enables Figma users to generate, review, and iterate on code directly from design assets without leaving the platform. This is a meaningful workflow compression: the gap between a finished design and a working front-end is now being collapsed by AI coding agents operating inside the tool where the design was created. The back-to-back integrations with both Anthropic and OpenAI suggest Figma is pursuing a model-agnostic strategy to remain the dominant design-to-development handoff layer as AI coding tools restructure the product development pipeline.

## Key Data Points

- Figma integrated Anthropic's Claude Code and OpenAI's Codex in close succession — within days of each other
- Codex is OpenAI's purpose-built code generation model, distinct from general ChatGPT capabilities
- The integration allows code generation directly from Figma design assets within the same workspace
- Competing design-to-code AI tools include Builder.io, Locofy, and GitHub Copilot for Visual Studio integrations
- Figma's strategy appears model-agnostic — supporting multiple AI providers rather than committing to one
- This follows broader industry consolidation where design, coding, and AI tooling pipelines are merging

## Why It Matters

The design-to-code handoff has historically been one of the most friction-heavy steps in product development — responsible for miscommunication, rework, and delays. AI coding agents embedded directly inside Figma effectively eliminate that handoff for many use cases. For enterprise product teams, this is not a feature update; it is a workflow restructuring. The fact that Figma is integrating both Claude Code and Codex signals that no single AI coding model has won that market yet — and that the platform layer, not the model layer, may be where the durable value accumulates. Developers and designers evaluating AI coding tools should be watching platform integrations as closely as model benchmarks.
