## Summary
n8n has featured a workflow template that pulls KPI metrics directly from a Google Sheets document, transforms them into structured JSON, and generates a fully interactive static HTML dashboard — complete with KPI cards, charts, and data tables — hosted as a self-contained web page. The workflow can be triggered on demand or run on a schedule, enabling auto-updating dashboards without any BI tool subscription or dedicated data infrastructure. The solution uses n8n's CustomJS node for data transformation and HTML generation, making the entire pipeline configurable without writing backend code. It eliminates the need for tools like Tableau, Power BI, or Looker for teams that live in Google Sheets. The output is a live, shareable HTML page that updates automatically whenever the workflow runs.
