# Perplexity Launches "Computer" — An AI Agent That Orchestrates Other AI Agents

**Source URL:** https://arstechnica.com/ai/2026/02/perplexity-announces-computer-an-ai-agent-that-assigns-work-to-other-ai-agents/

**Publication Date:** February 26, 2026

## Summary

Perplexity has launched "Computer," an agentic AI system that decomposes user goals into subtasks and assigns each to whichever AI model it judges best-suited for that job. Rather than routing all tasks to a single model, Computer functions as a meta-orchestrator: it uses Anthropic's Claude Opus 4.6 for core reasoning, Gemini for deep research, Nano Banana for image generation, Veo 3.1 for video, Grok for lightweight speed tasks, and ChatGPT 5.2 for long-context recall. The system runs entirely in the cloud in isolated compute environments with a real filesystem, browser, and tool integrations — and is designed to run for "hours or even months" on complex workflows. Currently available to Perplexity Max subscribers, Computer is a direct response to OpenClaw (the viral but chaotic agentic tool that mass-deleted emails and launched phishing attacks) — offering a more constrained, walled-garden version of the same concept. Perplexity frames it as the App Store to OpenClaw's open web.

## Key Data Points

- Core reasoning engine: Anthropic Claude Opus 4.6
- Task-specific model routing: Gemini (deep research), Nano Banana (image generation), Veo 3.1 (video), Grok (lightweight/speed), ChatGPT 5.2 (long-context/search)
- Runs in the cloud in isolated compute environments (not on user's local machine)
- Designed to run "hours or even months" on complex multi-step workflows
- Available now to: Perplexity Max subscribers
- Runs with a real filesystem, real browser, and real tool integrations
- Positions against: OpenClaw (viral, uncontrolled), Claude Cowork (single-model)
- Key distinction from competitors: best-model-for-subtask routing across multiple AI labs' models
- OpenAI reportedly hired OpenClaw's developer; Sam Altman indicated OpenClaw-style capabilities are core to OpenAI's product vision
- The system curates integrations (walled garden approach) to prevent the instability seen with OpenClaw plugins

## Why It Matters

Perplexity Computer represents a strategic inflection in how enterprise agentic AI is being positioned: not a single powerful model, but a system that directs an ensemble of the best available models. This multi-model orchestration approach — best tool, best task — is how large-scale knowledge work will actually be automated. The deliberate "walled garden" framing also signals that the agentic AI market is already bifurcating between raw capability (OpenClaw) and governed capability (Computer, Copilot Tasks). For enterprises evaluating AI workflow tools, Computer is an early commercial embodiment of what "agentic AI in production" might look like: complex, multi-step, multi-modal, cloud-isolated, and operating without constant human intervention.
