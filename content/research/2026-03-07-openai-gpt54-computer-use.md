# Research: OpenAI Launches GPT-5.4 — First Model with Native Computer Use

**Title:** OpenAI's new GPT-5.4 model is a big step toward autonomous agents

**Source URL:** https://www.theverge.com/ai-artificial-intelligence/889926/openai-gpt-5-4-model-release-ai-agents

**Publication Date:** March 5, 2026

**Keyword Match:** AI agents — enterprise adoption (#2); OpenClaw / agentic AI tools (#1)

---

## Summary

OpenAI has launched GPT-5.4, its most capable frontier model for professional work and the company's first model with native computer use capabilities built in. Unlike previous models that could only communicate through text, GPT-5.4 can operate a computer on a user's behalf — issuing keyboard and mouse commands in response to screenshots, writing code to control applications, and completing tasks across different software environments without constant human prompting.

The model ships in three tiers: GPT-5.4 (base, available in the API and Codex), GPT-5.4 Thinking (a reasoning model rolling out to ChatGPT Plus, Team, and Pro users), and GPT-5.4 Pro (maximum performance for complex tasks, available in the API and for Enterprise and Edu ChatGPT users). OpenAI positions GPT-5.4 as the most factual model it has released to date — individual claims are 33% less likely to be false compared to GPT-5.2.

The computer use capability is the headline feature. GPT-5.4 can write code to operate computers, issue keyboard and mouse commands in response to screenshots, and is improved at using web browsers and calling external tools and APIs. It can also search across multiple rounds to synthesize answers to "needle-in-a-haystack" questions from multiple sources. The GPT-5.4 Thinking model shows its reasoning outline to users and allows them to steer or redirect its work mid-response — reducing the need to start over.

The launch arrives amid significant controversy over OpenAI's recently announced deal with the US Department of Defense, which some OpenAI employees and external observers have criticized as enabling the types of AI military use that Anthropic had refused to support.

## Why It Matters

Native computer use in a frontier model is the threshold moment that separates "AI that answers questions" from "AI that does work." GPT-5.4's ability to operate applications, browse the web, and execute tasks across software environments makes it the clearest instantiation yet of the agentic AI paradigm that analysts have been predicting. For enterprise buyers and developers, this changes the cost-benefit analysis of AI deployment: an agent that can interact with existing software infrastructure without requiring custom integrations is orders of magnitude more deployable than one that cannot. The 33% factuality improvement is also significant for enterprise use cases where accuracy is non-negotiable. This is the model that makes "from pilot to production" a realistic near-term goal for many organizations.
