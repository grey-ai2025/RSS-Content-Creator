# Anthropic Launches Code Review Tool to Manage the Vibe Coding Flood

**Title:** Anthropic launches code review tool to check flood of AI-generated code

**Source URL:** https://techcrunch.com/2026/03/09/anthropic-launches-code-review-tool-to-check-flood-of-ai-generated-code/

**Publication Date:** March 9, 2026

**Keyword Match:** AI-powered code security; Agentic engineering; AI agents in enterprise

**Summary:** Anthropic launched Code Review inside Claude Code on March 9, targeting enterprise teams overwhelmed by the volume of pull requests that AI coding tools now generate. The product is a multi-agent system that automatically analyzes pull requests on GitHub, flags logic errors by severity, and leaves step-by-step explanations — directly addressing the bottleneck that vibe coding has created between code generation and code shipping. Rather than catching style issues, the tool focuses exclusively on logic errors, a deliberate choice to avoid the developer frustration that comes with non-actionable AI feedback. Enterprises including Uber, Salesforce, and Accenture are already among the target users. The launch arrives as Claude Code's run-rate revenue has surpassed $2.5 billion, with enterprise subscriptions quadrupling since the start of 2026.

**Key Data Points:**
- Claude Code's run-rate revenue has surpassed $2.5 billion since launch
- Enterprise subscriptions for Claude Code have quadrupled since January 2026
- Code Review integrates with GitHub and automatically analyzes pull requests
- Multi-agent architecture: multiple agents examine the codebase from different angles simultaneously; a final agent aggregates and ranks findings
- Severity labeling: red (highest), yellow (potential issues), purple (historical/preexisting bugs)
- Estimated cost: $15–$25 per review, token-based pricing
- Available first to Claude for Teams and Claude for Enterprise customers in research preview
- Named enterprise users: Uber, Salesforce, Accenture
- Anthropic's separately launched Claude Code Security provides deeper security analysis (distinct from Code Review's logic focus)
- Quote from Cat Wu, Anthropic head of product: "Enterprise leaders keep asking us: Now that Claude Code is putting up a bunch of pull requests, how do I make sure those get reviewed in an efficient manner?"

**Why It Matters:** The vibe coding wave has created an unexpected enterprise problem — code is being generated faster than it can be reviewed. This tool signals that AI productivity gains at the generation layer are creating bottlenecks at the review layer, and that a new category of AI-powered quality infrastructure is now necessary. For enterprise leaders, this reframes the ROI question: the cost of AI coding tools must now include the cost of AI reviewing those tools' output. The $15–25 per review price point will force a genuine calculation about whether speed gains justify the new overhead.
