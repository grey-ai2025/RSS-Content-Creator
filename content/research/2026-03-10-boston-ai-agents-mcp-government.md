# Boston's CIO Is Building AI Agents on MCP to Open Government Data to the Public

**Title:** AI agents are coming for government. How one big city is letting them in

**Source URL:** https://www.fastcompany.com/91504876/boston-cio-santi-garces-on-ai-agents-mcp-open-data

**Publication Date:** March 9, 2026

**Keyword Match:** Model Context Protocol (MCP); AI agents in enterprise; Agentic AI; AI governance and sovereignty

**Summary:** Boston's Chief Information Officer Santi Garces is running a pioneering experiment to deploy AI agents using Model Context Protocol (MCP) to give residents and city staff expanded access to government data and services. Rather than building a traditional AI chatbot, Boston is using MCP's standardized connectivity layer to let agents interact with multiple city data systems — enabling use cases that would have required months of custom integration work per system. The experiment positions government as an early adopter of the agent-plus-MCP stack, and raises important questions about public-sector AI governance, data transparency, and what happens when AI agents handle civic infrastructure. The article frames this as the leading edge of how governments will interact with their constituents in the agentic era.

**Key Data Points:**
- Boston CIO Santi Garces is spearheading the initiative
- MCP (Model Context Protocol) is the integration layer enabling agents to connect to multiple government data sources without bespoke API builds
- Goal: expand public access to government information through AI agent interfaces
- Use case framing: instead of searching government websites, citizens could query an AI agent that has access to city systems in real time
- The experiment represents one of the first documented public-sector deployments of the agent + MCP architecture
- Fast Company frames this as part of a broader "AI agents coming for government" trend
- Key tension: balancing open data access with data privacy and governance concerns in a public-sector context

**Why It Matters:** The MCP standard is rapidly becoming the connective tissue of the agentic enterprise — but most coverage focuses on corporate deployments. Boston's experiment shows the model extending into public infrastructure, which carries different stakes: government data touches housing records, permits, tax information, and services that citizens depend on. If MCP can simplify government data connectivity the way it has for enterprise tools, it represents a genuinely transformative shift in civic technology. It also signals that MCP is mainstream enough that government CIOs are now running pilots — moving from tech community standard to institutional adoption.

**Related Themes:** Model Context Protocol (MCP), AI agents in government, agentic AI, civic technology, AI governance, open data, enterprise AI infrastructure
