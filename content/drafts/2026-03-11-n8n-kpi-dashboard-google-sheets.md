## HEADLINE
A free n8n workflow template converts a Google Sheets KPI tracker into a live, self-hosted HTML dashboard — with cards, charts, and tables — without any BI tool subscription.

## KEY FACTS
- Google Sheets is the only data source required; no database, no warehouse, no BI platform
- Output: a fully styled static HTML page with KPI cards, charts, and summary tables, built with Tailwind CSS for responsive design
- Runs on demand or on a schedule — the dashboard auto-refreshes every time the workflow executes
- Source credibility: Official n8n featured workflow template, available free in the n8n library

## GREY AI ANGLE
This workflow is a concrete example of automation replacing a software category — not augmenting it. For teams spending on Tableau, Power BI, or Looker at the SMB level, the question shifts from "which BI tool?" to "do we need one?" That is an AI literacy and organizational readiness conversation. Teams that understand automation as infrastructure will cancel the subscription; teams that don't will keep paying.
