## HEADLINE
Enterprise subscriptions for an AI coding platform quadrupled in one quarter — forcing the launch of a separate AI product just to review all the code the first AI is generating.

## KEY FACTS
- Enterprise subscriptions for the platform quadrupled since January 2026; run-rate revenue surpassed $2.5 billion
- Code Review uses a multi-agent architecture: parallel agents examine the codebase from multiple angles simultaneously, then a final agent aggregates and ranks findings by severity
- Cost: $15–$25 per review, token-based — the first concrete figure forcing a ROI calculation on AI coding productivity claims
- Source credibility: Enterprise platform in production, named enterprise users include three Fortune 500 companies

## GREY AI ANGLE
The vibe coding wave created an invisible bottleneck: code generation accelerated, but the review layer stayed human-speed. This is the organizational readiness problem hiding inside every AI productivity story — speed at the generation layer creates a new constraint at the quality layer. The $15–25 per-review cost is the clearest signal yet that AI adoption ROI calculations must now account for the infrastructure required to govern AI outputs, not just produce them.

**Provocative:** AI is writing code faster than engineers can review it. So they built another AI to do the reviewing.

**Binary:** Vibe coding solves the generation problem. It created the review problem. Now you need a second AI to fix what the first AI caused.

**Stat Gap:** Enterprise AI coding subscriptions: quadrupled in one quarter. Code review capacity: unchanged. A $15 fix for a $2.5 billion bottleneck.
