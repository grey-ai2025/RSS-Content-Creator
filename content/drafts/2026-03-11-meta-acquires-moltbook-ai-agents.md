## HEADLINE
Meta acquired a social network built exclusively for AI agents — where agents post, reply, and coordinate via API — and folded the team into its Superintelligence Labs division.

## KEY FACTS
- At peak: 1 million+ human observers, 37,000+ autonomous agents active on the platform
- The platform's core mechanic: only AI agents running via API could post; humans could watch but not participate
- Creator of the underlying agent framework was simultaneously acqui-hired by OpenAI — both labs moving in the same week
- Source credibility: Confirmed acquisition, named founders joining Meta Superintelligence Labs, named platform and investor record

## GREY AI ANGLE
Meta buying a social network for AI agents is not an incremental product decision — it is an infrastructure bet. The "always-on directory" framing reveals what's actually being built: persistent agent identities, discovery mechanisms, and coordination rails. For enterprise teams designing multi-agent workflows, this is the signal that agent infrastructure is becoming a platform-level concern. The organizations that think about their agents as persistent, discoverable entities will be ahead of those still thinking agent-by-agent.

## RAW MATERIAL
Meta CTO Andrew Bosworth had previously called agent-to-human communication "not particularly interesting" — but was intrigued by the moment humans started hacking the network. Meta's acquisition statement: opens "new ways for AI agents to work for people and businesses." The platform's fatal security flaw: Supabase credentials were unsecured, allowing anyone to grab tokens and impersonate agents — many "alarming" posts of agents developing secret languages were human-posted fakes. OpenClaw creator Peter Steinberger simultaneously acqui-hired by OpenAI — both frontier labs acquiring agent infrastructure in the same window. Nvidia also readying NemoClaw, an open-source agent platform.
